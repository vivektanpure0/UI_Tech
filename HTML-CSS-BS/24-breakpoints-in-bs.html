Bootstrap Breakpoints

Boostrap defines 4 breakpoints and classifies devices into 4 based on viewport widths

  • Extra small device < 576px
  • 576px ≤ Small device < 768px;
  • 768px ≤ Medium device < 992px;
  • 992px ≤ Large device < 1200px
  • 1200px ≤ Extra Large device

Example: A .container gets different widths for different viewport widths
